Sbt Dependency Analyzer for IntelliJ IDEA GitHub | Issues

Adds support for the Scala's sbt. The following features are available for free with IntelliJ IDEA:
  • Show Dependency Tree
  • Show Conflicts
  • Search Dependencies
  • Goto Dependency
  • Show Dependencies Between Modules
  • Show JAR Size
  • Dependency Exclusion (Experimental)

Visit GitHub for instructions on how to properly use this plugin!
